
# Проект

Веб-приложение кофейни с административной панелью

# О проекте

Учебный проект по модернизации легаси-сайта кофейни до уровня полноценного веб-приложения с защищённым бэкендом, админ-панелью, базой данных, дизайном, обеспеченный безопасностью.

## 🛠 Стек технологий

- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, MySQL
